Программа поддерживает:

+ Вычисление f(x) в заданной точке
+ Построение таблицы значений на интервале
+ Поиск минимального и максимального значений на интервале
+ Вычисление значения x по y
+ Вычисление производной в точке методом центральных разностей
+ Сохранение таблицы значений в файл

# Структура программы:  
## Определения предпроцессора:  
_CRT_SECURE_NO_DEPRECATE  

## Подключаемые библиотеки:  
stdio.h — ввод/вывод, файлы  
locale.h — русская локализация  
math.h — математические функции  
float.h — параметры типов с плавающей точкой  

## Глобальные переменные:  
main()  
setlocale(LC_ALL, "RUS")  
Вывод описания кусочной функции f(x)  
Цикл do-while(choice != 0)  
Меню:  
>1. Значение функции в точке  
>2. Таблица значений  
>3. Минимум и максимум  
>4. Поиск X по Y  
>5. Производная в точке  
>6. Сохранить таблицу в файл  
>0. Выход  
switch(c) вызывает операции над функцией  

## Основная функция f(double x):
